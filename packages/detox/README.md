# Limrun Detox

Limrun integration helpers and a Detox third-party driver for running Detox
against apps on Limrun remote iOS simulators.

This package is coupled to Detox 20.x internals and should be versioned with
that compatibility in mind.

For a runnable Expo Go example, see `examples/detox-ios` in the
`limrun-inc/typescript-sdk` repository.
